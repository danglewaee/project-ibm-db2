What is actually proved here.

This page keeps the cloud story honest by separating live-demo proof from the earlier Db2 validation work.

Proved on the public deployment

  • The backend packages cleanly into a public Cloud Run service.
  • The root experience is no longer an exposed API wall; it behaves like a product front door.
  • The service still exposes live health and system metadata for inspection.

Proved earlier, not re-claimed here

  • Reservation, shipment, cancellation, reconciliation, and audit behavior were validated against live Db2 in Docker.
  • Flyway migrations and integration flows were exercised before the public Cloud Run demo was published.
  • The IBM Cloud Code Engine path exists in the repo, but this page does not pretend that paid-cloud path was completed.
